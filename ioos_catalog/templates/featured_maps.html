IOOS Featured Maps

AOOS Ocean Data Explorer

This portal contains scientific and management information including real-time sensor feeds, operational oceanographic and atmospheric models, satellite observations and GIS data sets that describe the biological, chemical and physical characteristics of Alaska and its surrounding waters.

CariCOOS Latest Conditions and Forecasts Map

The CariCOOS Latest Conditions and Forecasts Map graphically displays the latest observations and forcasts form the Caribbean Coastal Ocean Observing System (CariCOOS).

CENCOOS Data Portal

This map contains station data from the Californian coast, streaming in real-time with a history of up to seven days.

CORDC HF-Radar

An asset map of the coastal HF-Radar stations and their data output.

GCOOS Data Portal

This Data Portal provides timely information about the environment of the United States portion of the Gulf of Mexico and its estuaries for use by decision-makers, including researchers, government managers, industry, the military, educators, emergency responders, and the general public. Observing stations in the region are monitored constantly.

GLOS Data Portal

The GLOS Data Portal provides access to near-realtime and archived observations and to model forecasts for the Great Lakes.

MARACOOS Asset Map

The MARACOOS asset map displays gridded datasets from model output in conjunction station data, for the mid-atlantic region.

NANOOS Visualization System

The NVS contains a data explorer which has a mapping of the NANOOS assets and an in-situ data graph and data comparison with model output.

NDBC HF-Radar

This is a demonstration of the NOAA HF Radar National Server and Architecture Project. HF Radar is used to remotely measure ocean surface currents.

NERACOOS Real-time Data Portal

This map displays real-time observations from buoys and monitoring stations in the Northeast region.

PacIOOS Voyager

The PacIOOS Data Management System focuses on the management, service, and delivery of PacIOOS-related data, allowing PacIOOS to respond to partner needs for sharing and integrating data, offer decision-making products and access to data download sites, and to display information in real-time and forecasting formats.

SCCOOS Interactive Map

The SCCOOS Interactive Map has a listing of available products that can be displayed on an interactive map as well as an interface for all SCCOOS observations.

SECOORA Data Portal

The SECOORA asset map displays station information for the south east region with optional map-overlays of bathymetric data.
